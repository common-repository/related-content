== Description ==

Related content for posts and pages

<strong>Related Content is tested,<br />
if you find a bug please open a ticket in the [support request](https://wordpress.org/support/plugin/related-content).<br />
Every issue will be fixed asap!</strong>                       

**The Related Content plugin features:**

* FULLY FREE related content plugin<br />
* show related to post links<br />
* show related to page links<br />
* shortcode can be used to show related content links on page<br />
* shortcode can be used to show related content links on post<br />
* randomized related links for post, pages and categories<br />
* related content block with Title or without Title<br />
* no our copyrights<br />
* no third party links<br />
                                   
As "related" will be used the content from your own WordPress website and showed for content linking in block on your permitted pages.

= How to use Related Content plugin? =
After Related Content plugin was successfully activated,<br />
set the Setting for plugin,<br />
copy shortcode and past on any post or page of your website
                                                           
If you need help with the related content or you want to report a bug please refer to our support on www.wpadm.com

== Installation ==

1. Upload the plugin folder `related-content` to the `/wp-content/plugins/` directory of your WordPress installation;
2. Activate the plugin through the `Plugins` menu in WordPress;
3. Go to "Related Content" plugin page in your WordPress installation;
4. Copy the short code of plugin and Paste on any content post or content page of your WordPress website;

if you find a bug please open a ticket in the [support request](https://wordpress.org/support/plugin/related-content).<br />
